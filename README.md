# ReasonLabs-EPP-EDR-Removal-Workaround

I created this guide because I searched online and could not find a complete, step-by-step solution that actually worked.

This procedure was successful in my case, so I hope it helps others facing the same issue.

I decided to remove the software after detecting that it had begun establishing active connections to an endpoint with a poor reputation, along with other suspicious and unexplained network activity.

Please review the commands carefully and test them in a controlled environment before applying this procedure to a production system.

Important: Run Command Prompt and PowerShell as Administrator.

Only perform this procedure on systems that you own or are authorized to administer.
